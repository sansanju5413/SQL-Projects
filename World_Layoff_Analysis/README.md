# 🌍 World Layoffs Data Cleaning & Analysis

## 📌 Project Overview

This project focuses on cleaning and analyzing a global layoffs dataset using SQL (MySQL).

The objective was to transform raw data into a clean and reliable dataset, then perform exploratory data analysis (EDA) to uncover meaningful business insights related to layoffs across companies, industries, countries, and years.

---

## 📂 Dataset

- Raw Layoffs Dataset (.csv)
- Cleaned Layoffs Dataset (.csv)

---

## 🧹 Data Cleaning Tasks

- Removed duplicate records
- Standardized company and industry names
- Handled null and missing values
- Corrected inconsistent data formats
- Improved overall data quality for analysis

---

## 📊 Exploratory Data Analysis (EDA)

Performed analysis to identify:

- Layoffs by company
- Layoffs by industry
- Layoffs by country
- Year-wise layoff trends
- Companies with the highest layoffs
- Industries most affected by layoffs
- Relationship between funding and layoffs
